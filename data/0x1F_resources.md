### Essential Tools & Resources

**Operating Systems:**
*   **[Tails OS](https://tails.boum.org/)**: Live USB OS. Amnesic and Tor-enforced.
*   **[Qubes OS](https://www.qubes-os.org/)**: Security by compartmentalization.
*   **[Whonix](https://www.whonix.org/)**: Tor Gateway + Workstation VM combo.

**Encryption:**
*   **[VeraCrypt](https://www.veracrypt.fr/)**: Full disk encryption (successor to TrueCrypt).
*   **[GPG4Win](https://www.gpg4win.org/)**: PGP suite for Windows.
*   **[KeePassXC](https://keepassxc.org/)**: Offline password manager.

**Browsers & Network:**
*   **[Tor Browser](https://www.torproject.org/)**: The standard.
*   **[I2P](https://geti2p.net/)**: Alternative anonymous network.
*   **[Orbot](https://guardianproject.info/apps/org.torproject.android/)**: Tor for Android.

**Cryptocurrency:**
*   **[Electrum](https://electrum.org/)**: Lightweight Bitcoin wallet.
*   **[Feather Wallet](https://featherwallet.org/)**: Monero wallet (Tails-compatible).
*   **[Cake Wallet](https://cakewallet.com/)**: Mobile XMR/BTC wallet.

**Metadata Removal:**
*   **[MAT2](https://0xacab.org/jvoisin/mat2)**: Metadata Anonymisation Toolkit (pre-installed on Tails).
*   **[ExifTool](https://exiftool.org/)**: Command-line metadata editor.

**Communication:**
*   **[Pidgin + OTR](https://pidgin.im/)**: Encrypted instant messaging.
*   **[OnionShare](https://onionshare.org/)**: Anonymous file sharing.
